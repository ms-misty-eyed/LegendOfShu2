# LegendOfShu2

## Summary
This is a videogame that we made in the winter 2026 
semester at Champlain College.

The game is a first-person walking simulator. The goal is
to walk around an evolving forest and pick up objects
that represent memories of one's life. The choice to pick
up or go past them is yours.

## How to play
You can play in your browser on itch.io:
https://ms-misty-eyed.itch.io/legend-of-shu

## Software 
- Blender
- Visual Studio Code
- JetBrains Rider
- Unity
